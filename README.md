## 見積もり
### 難しい理由
- 曖昧さ
- 要件未確定
- メンバー未確定
- バッファの根拠
### 心構え
- システム要件一覧をもとに工数を推定
- 根拠をつくり、曖昧さをなくす
- 経験と勘
### 見積もり法
- FP(ファンクションポイント)法
### コストの見積もり
```
コスト = 単価 x 工数
```
- 原価なので、単価には利益やバッファも積む
